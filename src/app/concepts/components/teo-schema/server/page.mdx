# Server

Teo provides a unified way to specify server configurations.

## Server IP and port

Teo server's IP and port are binded through `server.bind` config item.

```teo showLineNumbers
server {
  bind: ("0.0.0.0", 5100)
}
```

## URL prefix

Teo server can have a URL path prefix. All request paths are prefixed with this 
prefix.

```teo showLineNumbers
server {
  pathPrefix "/v1"
}
```
