# Connector

The connector specifies which database to connect. Each 
[namespace](/concepts/components/teo-schema/namespace) can have zero or one 
connector. Teo currently supports MySQL, PostgreSQL, SQLite and MongoDB.

## Declaring

To connect to a database, specify a
[provider](/reference/api-reference/schema-reference#database)
<span className="concept">REFERENCE</span>
type and a connection URL.

```teo
connector {
  provider: .mysql,
  url: "mysql://127.0.0.1:3306/helloteo"
}
```

There are [examples](/reference/api-reference/schema-reference#examples) of how 
to specify a database URL in the 
[schema reference](/reference/api-reference/schema-reference).
