# Generate

The `generate` command generate 
[model entities](/concepts/components/teo-schema/generator#model-entity-generator)
<span className="concept">CONCEPT</span> and 
[query clients](/concepts/components/teo-schema/generator#query-client-generator)
<span className="concept">CONCEPT</span>.

## Model entity

Use the command `generate entity` to generate model entities.

<RustDoc>
```sh
cargo teo generate entity
```
</RustDoc>
<NodeJSDoc>
```sh
npx teo generate entity
```
</NodeJSDoc>
<PythonDoc>
```sh
teo generate entity
```
</PythonDoc>

## Query client

Use the command `generate client` generates query clients.

<RustDoc>
```sh
cargo teo generate client
```
</RustDoc>
<NodeJSDoc>
```sh
npx teo generate client
```
</NodeJSDoc>
<PythonDoc>
```sh
teo generate client
```
</PythonDoc>

Use the command `generate admin` to generate an admin dashboard.
