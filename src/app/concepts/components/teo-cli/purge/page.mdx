# Purge

The purge command is used to clear the database. This won't drop the database,
delete any tables. Instead, it just clear all existing records.

## Purging table data

Use the `purge` command to wipe out table data.

<RustDoc>
```sh
cargo teo purge
```
</RustDoc>
<NodeJSDoc>
```sh
npx teo purge
```
</NodeJSDoc>
<PythonDoc>
```sh
teo purge
```
</PythonDoc>
